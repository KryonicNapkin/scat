# scat
scat - simple cat 

Written in C11

## Installation

1. Clone this repository

```sh
git clone https://github.com/KryonicNapkin/scat.git
```

2. Build

```sh
cd scat
make
```

3. Usage

```sh
Usage: scat [FILE]
```
